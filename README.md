# SisCakes Web Menu 🍰

**Aplicação projetada para oferecer um cardápio online para uma confeitaria. Ela visa proporcionar uma experiência fluida para os clientes e um sistema eficiente de gestão para a loja.**

---

## 🔍 Visão Geral do Projeto 

O SisCakes Web Menu é um site criado para ajudar a confeitaria **SisCakes** a mostrar seu cardápio online de forma fácil e prática. Ele é feito para clientes que adoram bolos personalizados, fatias recheadas, bolos no pote e outras delícias, além de oferecer ferramentas para a loja gerenciar tudo isso. 

Vamos usar **Angular** para o front-end, **Java 21** com **Spring Boot** para o back-end, **PostgreSQL** para o banco de dados e **Docker** para organizar tudo em contêineres. O design segue a vibe da logo da SisCakes, com cores suaves, frutas e um estilo acolhedor, e funciona bem em celulares e computadores.

Iremos garantir que o site seja fácil de usar por todos (acessibilidade) e otimizado para buscadores (SEO). Além disso, o projeto foi pensado para crescer e se conectar com outros sistemas no futuro.

## 🛠 Funcionalidades

- Login e autenticação de usuário.
- Painel administrativo para gestão de cardápio e estoque.
- Atualizações em tempo real do cardápio.
- Integração com WhatsApp para pedidos dos clientes.
- Design responsivo para todos os dispositivos.
